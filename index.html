HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The head element contains information that is not visible to the user
  • The body element is known as the parent element
  • The nested elements within a body element are known as child elements
  • It is important to use semantic html elements, such as body, when possible as it makes your html easier to read by other developers and the browser
  • Visible content is contained between an opening tag and a closing tag

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git checkout main: takes you back to the main branch
  • git revert: command to revert to the previous version of code
  • It's important to work on tasks on feature branches first and then merge into the main branch
  • The main branch should be the final most updated version of your application
