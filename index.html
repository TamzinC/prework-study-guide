HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The head element contains information that is not visible to the user
  • The body element is known as the parent element
  • The nested elements within a body element are known as child elements
  • It is important to use semantic html elements, such as body, when possible as it makes your html easier to read by other developers and the browser
  • Visible content is contained between an opening tag and a closing tag

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Do Not Repeat Yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained
  • The CSS rule is made up of a selector (which defines the element or attributes to which the rules will apply) and declations (which contain two important components - the CSS property and value)
  • CSS determines the styles of a webpage in order to make a website visually appealing

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git checkout main: takes you back to the main branch
  • git revert: command to revert to the previous version of code
  • It's important to work on tasks on feature branches first and then merge into the main branch
  • The main branch should be the final most updated version of your application

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • You can create conditional statements giving one or more conditions for the computer to evaluate. If one condition is truthy the computer will execute the code provided between (), otherwise if it's false, the code won't execute, interrupting the control flow
  • for loops are often used to repeat a task and iterate over an array with only a few lines of code
  • Functions are reusable blocks of code that contain instructions to perform a specific task
  • You can comment out parts of code instead of deleting so you can come back to it later by using /* and */ at the beginning and end of the code
  • You can randomise a selection using the Math function
  • In order to execute a function, you must call it in your code
